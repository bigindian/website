How to Contribute
=================

```The Big Indian News``` needs you!

To contribute to the site and help keep our servers online donate to us directly using [PayPal](https://www.paypal.com/cgi-bin/webscr?cmd=_s-xclick&hosted_button_id=VVPBN75GCJ854). If you are looking to contribute in improving the code behind the site then keep reading.


Bugs & Suggestions
------------------
All bugs and suggestions are managed on GitHub using issues. Simple file an issue and give it the appropriate tag. Once it has been reviewed it will get merged into the develop branch where it get tested before going on to the master branch.


Guidlines
---------
There are really not much guidlines other than leaving as much as comments as you can! You are free to follow whatever coding standard you like :)
